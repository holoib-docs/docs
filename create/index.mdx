You can create Holoib interactive demos in two ways.

## Using the Holoib Chrome extension
Creating demos with our Chrome extension is incredibly simple and easy. It only takes 5 minutes to prepare your interactive demo. To begin, make sure to install our Chrome extension.

<Note> Pin the extension so that you can easily access it when needed. </Note>

## Create demos manually
Using the Holoib Chrome extension is the easiest and most convenient way to create demos. However, you can also create demos by uploading your own images.
